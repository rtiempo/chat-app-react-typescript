# Chat App

This is a chat app built using react, typescript and firebase.

## Getting Started

1. clone the project

```sh
git clone  https://github.com/rtiempo/chat-app-react-typescript.git
```

2. cd into the directory

```sh
cd chat-app-react-typescript
```

3. install dependencies

```sh
npm install
```

## Running the app

Development

```sh
npm start
```
